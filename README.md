## What are Agent Skills?

Agent Skills are folders containing a `SKILL.md` file with metadata and instructions that tell an agent how to perform a specific task. Skills can bundle scripts, templates, and reference materials to provide comprehensive guidance.

For more information, see the [Agent Skills documentation](https://agentskills.io/).
